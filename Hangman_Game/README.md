# Hangman Game

---

## 📌 Description

This is a console-based Hangman game built using Python. The player guesses a randomly selected word one letter at a time. Each wrong guess adds a part to the hangman drawing. The game ends when the player either guesses the word correctly or runs out of attempts.

---

## ⚙️ Features

- Random word selection
- ASCII-based hangman drawing
- Input validation (only single letters allowed)
- Tracks guessed letters
- Win and lose conditions

---

## 🛠️ Technologies Used

- PYTHON
- RANDOM MODULE

---

## ▶️ How to Run

- Run the program
```
python hangman.py
```

---

## 🎯 Object

- The objective of the game is to guess the hidden word correctly before the hangman is fully drawn. Each incorrect guess brings you closer to losing the game.
