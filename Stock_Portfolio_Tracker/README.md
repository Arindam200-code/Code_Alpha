# Stock Profolio Tracker

---

##📌 Description

The Stock Portfolio Tracker is a simple Python project that helps users manage and calculate their stock investments. It stores stock prices and quantities, calculates total investment value, and exports the data into a CSV file for easy analysis.

---

## 🚀 Features

- Store stock prices using Python dictionary
- Manage stock portfolio with quantity
- Calculate total investment automatically
- Export portfolio data to CSV file
- Simple and beginner-friendly automation project

---

## 🛠️ Technologies Used

- Python 3
- CSV module (built-in Python library)
- Dictionary
- Basic file handling

---

## ▶️ How to Run

- Run the program
```
python Stock_Tracker.py
```
